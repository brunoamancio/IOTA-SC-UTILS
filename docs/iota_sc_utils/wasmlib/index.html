Module iota_sc_utils::wasmlib[−][src]

Expand description

Integrated tool responsible for communication with ISCP’s vm. This is a modified version of the original IF’s wasmlib. Differences are kept minimal and only exist if they provide simpler interface, nothing else. Documentation: <https://github.com/iotaledger/wasp/tree/master/contracts/rust/wasmlib/docs> Github repository: <https://github.com/brunoamancio/IOTA-SC-HName-Generator>
